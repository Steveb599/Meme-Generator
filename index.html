Welcome to Steve's Meme Generator Project!

This project is a fun and interactive web application that allows users to create custom memes with their own captions and share them with the world. As a student at Coding Academy, I developed this project using web technologies such as HTML, CSS, and JavaScript. The project utilizes a canvas element to render the memes and provides users with various options to customize their memes. Users can choose from a collection of images, change the font size and color, add multiple captions, and download the memes to share them with their friends and family.
